Автозапуск авто на arduino и модеме m590e

Устройство предназначено для запуска авто с автоматической коробкой передач
Команды на запуск авто с авторизованного телефона
Вариант 1: СМС команда Start
Вариант 2: 2 звонка на номер с перерывом не более 30 секунд

Установить (прописать) авторизованный номер можно отправив СМС формата "P0 79001112233", в рамках безопасности это может сделать или уже авторизованный номер, или при нажатом тормозе и выключенном двигателе

Все СМС команды 

// автозапуск
// запуск возможен двойным звонком на номер сим карты, оба звонка должны быть сделаны на 30 секунд
// если двигатель запущен, то одиночный звонок продлит время работы
// All commands:  - перечень СМС
// Start          - запуск
// Stop           - остановка
// Balance        - баланс встроенной сим карты
// Info           - краткая информация о системе (статус, напряжение, температура...)
// List           - перечень прописанных телефонов
// P0..2 phone    - установка прописанных номеров, формат СМС "P0 79001112233", в рамках безопасности это может сделать или уже авторизованный номер, или при нажатом тормозе и выключенном двигателе 
// SmsON          - включение уведомления о запуске, остальное смысловое общение, естественно через СМС
// SmsOFF         - выключение уведомления о запуске
// FactoryReset   - сброс на типовые настройки
// Time 0..30     - время прогрева от 1 до 30 минут, 0 - автоматический режим
// STms 0..3000   - время прокрутки стартера (в микросекундах) от 1 до 3000 микросекунд, 0 - автоматический режим
// Help           - перечень команд
